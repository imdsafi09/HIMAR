# HIMARE

**HIMARE: Human-Informed Map Augmentation with Panoramic LIDAR Imagery for Robotic Search and Rescue in Challenging Environments**

---

## Overview

HIMARE is a novel framework designed to enhance robotic search and rescue (SAR) operations in disaster-affected or challenging environments. The framework integrates human-informed data with panoramic LiDAR imagery to create augmented maps, providing critical insights into environmental conditions and enabling precise localization of individuals. HIMARE leverages advanced detection, tracking, and mapping techniques to improve situational awareness and facilitate efficient SAR missions.
