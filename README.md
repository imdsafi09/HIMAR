# HIMARE

**HIMAR: Human-Informed Map Augmentation with Panoramic LIDAR Imaging for Enhanced Robotic Search-and-Rescue**

---

## Overview

HIMAR is a novel framework designed to enhance robotic search and rescue (SAR) operations in disaster-affected or challenging environments. The framework integrates human-informed data with panoramic LiDAR imagery to create augmented maps, providing critical insights into environmental conditions and enabling precise localization of individuals. HIMARE leverages advanced detection, tracking, and mapping techniques to improve situational awareness and facilitate efficient SAR missions.
